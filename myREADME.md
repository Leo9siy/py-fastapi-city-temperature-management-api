# City and Temperatures

#### I use FreeWeatherAPI for update weather
#### Temperature can be None

## Before start, do this steps
````
1. Clone repository
2. Create Environment
3. Install all requirements
4. pip install -r requirements.txt
````

## Now you`re ready
````
1. Run uvicorn server
2. Go to localhost:8000/cities
3. Add some cities
````
## Update temperatures
````
1. Go localhost:8000/temperatures/update
2. Go localhost:8000/temperatures
3. Proffit
````
